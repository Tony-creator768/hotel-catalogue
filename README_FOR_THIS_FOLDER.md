# Upload this folder to GitHub Pages

Upload the CONTENTS of this folder to your GitHub repository root.

Required files for the website:
- index.html
- styles.css
- app.js
- config.js
- sample-data.js
- .nojekyll

Optional folder:
- images/ — use this if you want to store images in GitHub and reference them in Google Sheets as `images/file-name.jpg`.

Before publishing, open `config.js` and replace the placeholder Apps Script URL with your deployed Google Apps Script Web App URL.
